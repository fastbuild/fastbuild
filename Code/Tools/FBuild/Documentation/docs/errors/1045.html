1045 - Extraneous token(s) following '%s' directive.

Description
Extra tokens were found after what would otherwise be a valid directive.
Example
Config:
#if THING ExtraneousThing #endif
Output:
c:\test\fbuild.bff(1,11): FASTBuild Error #1045 - Extraneous token(s) following 'if' directive. #if THING ExtraneousThing ^ \--here
Fix:
#if THING #endif
