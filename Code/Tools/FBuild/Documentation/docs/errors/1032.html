1032 - Unable to open include '%s'.

Description
The specified include file could not be opened.
Example
Config:
#include "missing.bff"
Output:
c:\test\fbuild.bff(1):(11) FASTBuild Error #1032 - Unable to open include 'missing.bff'. #include "missing.bff" ^ \--here
Fix:
#include "notmissing.bff"
