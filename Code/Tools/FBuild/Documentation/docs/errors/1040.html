1040 - #undef of built-in token not allowed.

Description
Built-in tokens cannot be undefined.
Example
Config:
#undef __WINDOWS__
Output:
C:\Test\fbuild.bff(1,8): FASTBuild Error #1040 - #undef of built-in token not allowed. #undef __WINDOWS__ ^ \--here
