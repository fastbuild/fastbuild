1002 - Matching closing token %c not found.

Description
The terminating token for a construct (e.g. a String, Array or Struct) was not found.
Example
Config:
.String = 'Unterminated
Output:
C:\test\fbuild.bff(1,11): FASTBuild Error #1002 - Matching closing token ' not found. .String = 'Unterminated ^ \--here
Fix:
.String = 'Terminated'
