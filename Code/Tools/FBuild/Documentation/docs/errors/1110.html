1110 - Expected argument block following function call.

Description
A function is invoked using its previously defined name followed by a round brace enclosed argument list (or an empty pair of braces if there are no arguments).
Example
Config:
function Func(){} Func=
Output:
C:\test\fbuild.bff(2,5): FASTBuild Error #1110 - Expected argument block following function call. Func= ^ \--here
Fix:
function Func(){} Func()
