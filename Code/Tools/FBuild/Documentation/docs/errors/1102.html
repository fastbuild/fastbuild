1102 - '%s' ('%s') is of unexpected type '%s'. Expected '%s'.

Description
The dependency of a Function was not of the expected type.
Example
Config:
.Compiler = 'cl.exe' .CompilerOptions = '%1 %2' .CompilerOutputPath = 'tmp/' .Librarian = 'lib.exe' .LibrarianOptions = '%1 %2' Library( 'lib1' ) { .CompilerInputPath = 'Code/' .LibrarianOutput = 'lib1.lib' } Library( 'lib2' ) { .LibrarianOutput = 'lib2.lib' .CompilerInputUnity = 'lib1' // Should be a Unity }
Output:
c:\Test\fbuild.bff(11):(1) FASTBuild Error #1102 - Library() - 'CompilerInputUnity' ('lib1') is of unexpected type 'Alias'. Expected 'Unity'. Library( 'lib2' ) ^ \--here
Fix:
.Compiler = 'cl.exe' .CompilerOptions = '%1 %2' .CompilerOutputPath = 'tmp/' .Librarian = 'lib.exe' .LibrarianOptions = '%1 %2' Unity( 'unity' ) { .UnityInputPath = 'Code/' .UnityOuptutPath = 'out/' } Library( 'lib' ) { .LibrarianOutput = 'lib2.lib' .CompilerInputUnity = 'unity' }
