1042 - Unknown operator '%s'.

Description
An unknown operator was found while parsing an #if directive.
Example
Config:
#if bad(X)
Output:
C:\Test\fbuild.bff(1,5): FASTBuild Error #1042 - Unknown operator 'bad'. #if bad(X) ^ \--here
Fix:
#if file_exists("file.bff")
