1015 - Unknown Function.

Description
A function style token was encountered, but the name does not correspond to any built-in or user-defined Function.
Example
Config:
InvalidFunction() { }
Output:
c:\Test\fbuild.bff(1):(1) FASTBuild Error #1015 - Unknown Function. InvalidFunction() ^ \--here
Fix:
Alias( 'MyAlias' ) // A valid built-in Function { ... }
