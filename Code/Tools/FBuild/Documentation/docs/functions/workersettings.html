WorkerSettings

FBuildWorker .settings file format. Describes the settings for the FBuildWorker.

WorkerSettings { // General .FileVersion // Version of this worker .settings file .WorkMode // (optional) Worker work mode, default: 'when idle' // 'disabled' means do not run jobs // 'when idle' means only run jobs when cpus idle // 'dedicated' means run jobs when received .NumCPUsToUse // (optional) Number of CPUs for worker to use // default: half of the available cpus .StartMinimized // (optional) Start worker minimized, default: false // Sandbox .SandboxEnabled // (optional) Enable worker execution sandbox, default: false .SandboxExe // (optional) Worker sandbox executable, default: empty string // Example Windows sandbox tool: // https://github.com/iceb0y/windows-container .SandboxArgs // (optional) Worker sandbox args, default: empty string .SandboxTmp // (optional) Worker sandbox tmp dir, default: 'sandbox' // For now, FBuildWorker only sets low integrity // permissions for the dir on Windows systems. // Tags .WorkerTags // (optional) List of worker tags // default: empty list // example: // { // 'TopDownMemory', // 'OS=Win-7-64' // } }
