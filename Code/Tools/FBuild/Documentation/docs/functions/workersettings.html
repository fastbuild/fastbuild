WorkerSettings

FBuildWorker .settings file format. Describes the settings for the FBuildWorker.

WorkerSettings { // General .FileVersion // Version of this worker .settings file .WorkMode // (optional) Worker work mode, default: 'when idle' // 'disabled' means do not run jobs // 'when idle' means only run jobs when cpus idle // 'dedicated' means run jobs when received .NumCPUsToUse // (optional) Number of CPUs for worker to use // default: half of the available cpus .StartMinimized // (optional) Start worker minimized, default: false }
