Contact and Support

Documentation

FASTBuild documentation can be found here:

  • Online (via the website)
  • Offline (download documentation with source code)

Bugs Reports, Feature Requests or Questions

Open issues are tracked on GitHub. You can access the knowledge of the FASTBuild community there, or see if your question has already been answered. Publicly asking your question may help others too.

This is the best place to raise bug reports, feature requests or to ask general questions.

  • GitHub Issues
Contribute to FASTBuild

If you'd like to propose a change for integration into FASTBuild, you can do so via GitHub.

  • GitHub Pull Requests

Private Support

If the documentation or GitHub doesn't help, or your question can't be asked publicly, you can send an email for private support.

Please note that responses may be significantly delayed as support load is highly variable!
